1. MENCY LOBO

Comprehensive UX Roadmap

Background: Provided the most detailed, structured feedback in a formal roadmap document format.

Phase 1: Understand the Existing Product

1.1 UX Audit

  • ‚òê Audit all existing tools (SPARK, WAVE, POPCORN, RSD Meter) using Heuristic Evaluations
  • ‚òê Map user flow (start ‚Üí end) for each tool and note decision points, errors, frictions, dropoffs
  • ‚òê Check for UI consistencies across all platforms (web, mobile, getdose.app website)

1.2 User Research

  • ‚òê Conduct qualitative interviews with real users to explore pain points, mental models, concerns, unmet needs
  • ‚òê Conduct usability testing to explore shortcomings in the tools
  • ‚òê Conduct competitive analysis to understand what other platforms offer

1.3 Defining Success Metrics

  • ‚òê Define metrics for each toolkit (SPARK, WAVE, POPCORN, RSD Meter)
  • ‚òê Collaborate with experts (Steev) on metric definition
  • ‚òê Use metrics to measure design changes

Phase 2: Problem Definition, Strategy & Vision

Problem Definition

  • ‚òê Clarify user personas and define problems
  • ‚òê Redesign user flows/journey to address main concerns

Value Proposition

  • ‚òê Determine core value proposition per tool and how they interconnect
  • ‚òê Map userflow within the app

Guiding Principles

  • ‚òê Define UX principles: "Minimize friction at emotional stress points"
  • ‚òê "Give users a feeling of safety and control"
  • ‚òê "Progress is visible and motivating"

Phase 3: Design, Iterate & Test

3.1 Brainstorming

  • ‚òê Sketch alternate userflows, onboarding, homepage, navigation
  • ‚òê Focus on transitions between tools
  • ‚òê Brainstorm "emotional safety" flows (RSD)
  • ‚òê Ideate new features to support existing toolkit

3.3 Design Details & UX Copy

  • ‚òê Ensure states, error feedback, confirmations are clear, gentle, supportive
  • ‚òê For RSD: microcopy must be empathetic and calming
  • ‚òê Animations/transitions should be subtle and comforting
  • ‚òê Provide "safe exits" for overwhelmed users
  • ‚òê Use language users understand (not psychological jargon)

3.4 Design System & Component Library

  • ‚òê Define colors and typography (gentle, supportive, empathetic)
  • ‚òê Build shared design system
  • ‚òê Ensure consistency across mobile, web, desktop
  • ‚òê Document usage guidelines and accessibility standards

Prioritization - Immediate High-Impact Opportunities

Priority 1: Strengthen Brand and Product Connection

Issue: Disconnect between DOSE app and DamnHealthyDose brand

  • ‚òê Align visuals, tone, messaging across all touchpoints
  • ‚òê Highlight shared mission and story on website
  • ‚òê Create consistent brand style

Priority 2: Clarify "An OS for ADHD"

Issue: Tagline lacks clear, relatable explanation

  • ‚òê Explain "OS" in plain, emotional language
  • ‚òê Use visuals/animations showing how modules work together
  • ‚òê Communicate real-world benefits

Priority 3: Revamp the Website

Issue: Lacks consistent visual identity

  • ‚òê Redesign for consistency, readability, trust
  • ‚òê Emphasize credibility with user testimonies
  • ‚òê Utilize visuals to make website engaging

2. SHREYA

AI & Interaction Enhancement

1. Conversational AI

"Integrating conversational AI to make reflection more natural and interactive could reduce typing fatigue and help users externalize their thoughts in real time. This approach would make the process feel more guided, empathetic, and engaging."
  • ‚òê Integrate conversational AI for SPARK tool
  • ‚òê Reduce typing fatigue
  • ‚òê Make reflection feel guided and empathetic
  • ‚òê Enable real-time thought externalization

2. Emotion-Centered Interaction

"Adding more visual cues would help users express feelings quickly and effortlessly."
  • ‚òê Add visual emotion selection (icons, sliders, emotion wheels)
  • ‚òê Replace text input with visual cues
  • ‚òê Make emotion expression quick and effortless

3. Guided Progress & Check-ins

"Introducing structured guidance throughout the exercises with gentle reminders and progress check-ins could help maintain consistency, accountability, and emotional support throughout their journey."
  • ‚òê Add structured guidance throughout exercises
  • ‚òê Implement gentle reminders during sessions
  • ‚òê Add progress check-ins
  • ‚òê Provide emotional support throughout process

4. Personalized Action Plans (with AI)

"AI could be used (with caution) to generate personalized action plans based on their responses. This would reduce decision fatigue while ensuring that suggested actions align closely with each user's emotional state."
  • ‚òê Use AI to generate personalized action plans
  • ‚òê Reduce decision fatigue
  • ‚òê Align actions with emotional state
  • ‚òê Maintain user control and safety

5. POPCORN Media Enhancement

  • ‚òê Enable image attachments
  • ‚òê Add audio recording capability
  • ‚òê Allow video attachments
  • ‚òê Enable link/URL saving

6. Branding & Color Refinement

"The current palette gives off more of a 'tech startup' feel. Using a calmer, more inviting visual identity could build stronger trust and improve user conversion."
  • ‚òê Refine color palette to be calmer and more inviting
  • ‚òê Move away from "tech startup" aesthetic
  • ‚òê Build stronger trust through visual identity
